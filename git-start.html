Steps to start with git repostitory

There are two ways to work on git repository.

  1. Clone exsisting repository using Git Desktop application which is UI based.
  2. Copy URI porvided by git repository and clone using git/bash terminal commands.

      3. We will discuss second type and strat work from your local server (computer), then push (uploading) your code to Git repository (remote server)

      1. Open a new folder where you want to create your local server copy.
      2. Open terminal at this folder
      3. git init
      4. git clone(paste https url)
      5. starting developing
      6. When you feel its time to save your work to local server, run commit command
      7. git add .
      8. git commit -m "your comment on commit"
      9. Finally after commit,its time to push your code to github repository
      10. git push origin master

      When you want to work on 'Branch'(copy of master). Create branch git branch bebugMe do commit/push on branch named bebugMe the merge your code to master

      1. git checkout master
      you can generate pull request on master to merge code. git pull git push
