Git branch created using below steps

  1. git branch bug_rm
  2. git checkout bug_rm
  3. --Edit your changes--
  4. git commit -m "add new feature and bug fix"
  5. git checkout master
  6. git merge bug_rm
  7. --As you have merge branch to master. Now its time to delete branch "bug_rm"--
  8. git branch -d bug_rm
  9. git push
